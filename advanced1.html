Example 1

This is an example about the mean weight of bread. It contains two numeric, two single choice and one multiple choice question.

It uses several add-ons described here, such as:

  • including a picture
  • creating and providing a .csv-file containing the data for direct download within the exercise
  • data sampling and inline code
